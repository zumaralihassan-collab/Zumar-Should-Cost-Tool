# Should Cost AI

AI-powered procurement should-cost calculator for GSM and sourcing teams.

**Live demo:** `https://<your-username>.github.io/should-cost-ai/`

## What it does
Given a product name (e.g. *"red sand brick"* or *"perfume 50 ml"*), the app classifies the product and proposes every relevant cost driver — raw materials, packaging, labor, energy, equipment, facility, overhead, QC, logistics, and supplier margin. Defaults are auto-adjusted by region (India / China / Vietnam / EU / US…), production volume, and quality tier. Every parameter is editable, with live charts, sensitivity sliders, scenario comparison, and Excel export.

## Built-in product templates
Red sand / fly-ash brick · Perfume EdT 50 ml · Cotton t-shirt · 4-layer PCBA · PET bottle · Generic skeleton

## Tech
Single self-contained HTML file. No build step. Loads Tailwind CSS, Chart.js, and SheetJS from CDN.

## Local use
Just open `index.html` in any browser.

## Deploy
The whole site is one file. Push to GitHub and turn on Pages — done.
